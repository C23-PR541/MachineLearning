# Deep Learning Model for Classyfing Workout Type
### Features for classifying
<ol>
  <li>Gender
    <ol>
      <li>Female : 0</li>
      <li>Male   : 1</li>
    </ol>
  </li>
  <li>Age</li>
  <li>Workout Experience (years) <int></li>
  <li>Workout Time  (minutes) <int></li>
  <li>Weight (kg) </li>
  <li>Height (cm) </li>
  <li>BMI</li>
</ol>
